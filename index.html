Installing WSL

If you have any questions/problems, ask me.

Turning the Windows features on

To add WSL as a feature, first press windows button
Then, type "Turn Windows features on or off" and press enter.
Next, check Windows Subsystem for Linux.
Restart if prompted

Installing and updating WSL

To install and update WSL, first press windows button
Next, type "Command Prompt" and press enter.
Then, type wsl --install and press enter.
Next, type wsl --update and press enter.

Installing Arch

To install Arch, press windows button
Then, type "Microsoft Store" and press enter
Next, search for "Arch WSL".
Then, click "Arch WSL" Next, click "Install"
